DIBS

Dictionary of Interacting Binary Stars

A dictionary of astronomical sources built for experts and beginners

X-ray binaries, cataclismic variables, neutron stars, black holes, white dwarfs, and more...

The field of interacting binary stars is large and growing, which means that it has developed a large lexicon with sometimes confusing terminology. DIBS is designed to gather some of the terminology used in the field of interacting binaries and explain it in a simple way. DIBS is also meant to be a community build webpage where anyone can submit or edit an article! Edit from Github Repository
